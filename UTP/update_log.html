Version 1.3_API
  • Added a free API to allow untrackable searches directly from the browser URL. Read below for more information.
  • API: Adding the "bingSearch" URL Search Parameter to the URL (e.g. {FilePath}/{FileName}.html?bingSearch=loan+sharks). This is not intended to be typed out in the URL but instead in the Custom Search Engine feature on Chrome or on an external website.
  • API: To use BuBu Untrackable Pages as a search engine (Chromium browsers only), go to chrome://settings, click "Search Engine", click "Manage" and under "Site Search" click "Add" and customize the fields. On the last field, paste the link of this page followed by the following string: `?bingSearch=%s` without the backticks. To search, type the inputted shortcut and press space then type your search. When BuBu Untrackable Pages opens, click on the word "Singapore" then press Alt + X (not Z)
  • 1.3.1: Added the ability to instantly open SLS (or any other website specified), and remove BuBu Untrackable Pages from your tab history by pressing Alt + S, Alt + L or Alt + Comma (,). You may have to press enter or click OK in the prompt after using the keyboard shortcut if you use this while browsing with BuBu Untrackable Pages.
  • 1.3.1: Removed update log from software launch page, significantly decreasing the software file size.
